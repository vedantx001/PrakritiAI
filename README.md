# 🌿 PrakritiAI

PrakritiAI is an **AI-Powered Ayurvedic Platform** built to make Ayurvedic wellness guidance more accessible, structured, and interactive through modern technology.

It combines **AI symptom analysis**, **Ayurvedic knowledge content**, **community engagement**, and **user contribution systems** into one ecosystem.

---

## 🚀 Live Demo

🔗 Live Project: https://prakriti-ai-gold.vercel.app/ 

---

## ✨ Features

### 🧠 AI Symptom Analysis
Users can enter their symptoms and receive a structured Ayurvedic response including:

- Immediate actions
- Possible dosha imbalance (Vata / Pitta / Kapha)
- Herbal remedies
- Ayurvedic therapies
- Lifestyle recommendations
- Diet suggestions
- Additional wellness notes

### 👤 User System

- Secure Authentication
- Personal Dashboard
- Health History Tracking
- Saved Content
- Contribution Records

### ⏳ Usage Control

- Logged-in users get full experience
- Guest users can analyze symptoms once every 24 hours

### 📚 Ayurveda Article Hub

Structured content format:

Series → Chapters → Articles

Users can:

- Read articles
- Like content
- Save content
- Share content
- Comment on content

### 🤝 Contribution System

Users can help improve the platform by:

- Suggesting corrections to existing articles
- Submitting new articles

Admin can:

- Accept contributions
- Reject contributions
- Publish approved changes live
- Give contributor credit/shoutout

### 💬 Community Discussion

Users can create posts, interact, and engage with others through:

- Likes
- Comments
- Saves
- Shares

### 🏆 Gamification

- Reward points system
- Top Contributors leaderboard

### 🛠️ Admin Dashboard

- Manage contributions
- Monitor platform stats
- Control articles and community ecosystem
